== Installation ==

1. Make sure you have the WooCommerce plugin installed and active;
2. Install clinicsoftware-woocommerce to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

Your connection and more info you can find on Status page.

Works with the latest WooCommerce.
